# 象棋 AI 分析講解 v0.3.0

v0.3.0 將專案整理為可安裝、可更新、可持續驗證的 Windows 桌面 MVP。

主要內容：

- 重整 task-first 操作介面、棋盤工具、即時分析與設定資訊架構。
- 支援 UCI／UCCI 多引擎登錄、主引擎與複核引擎，以及不平均分數的雙引擎裁決資料流。
- 支援 Anthropic、OpenAI、Gemini 與多種 OpenAI Chat Completions 相容服務。
- 加入有固定預算的 AI 解說品質迴圈、本機象棋知識、證據關聯與失敗區塊重寫。
- 加入資料備份／合併還原、錯題本、待理解局面、猜著紀錄與多輪追問。
- 完成 generic HTTPS 自動更新封裝、GitHub Actions CI 與可重複的 Release workflow。

驗證基線：typecheck、438 項測試、dependency audit、production build 與 Windows NSIS 封裝。

## 安裝

下載 `xiangqi-analyzer-0.3.0-setup.exe`，可直接覆蓋既有使用者安裝。使用者資料與已加密的 API Key 保存在個人資料目錄，不包含在安裝檔內。

## 已知限制

這個 v0.3.0 過渡安裝檔尚未使用受信任 CA 的 Windows 程式碼簽章，Windows SmartScreen 可能顯示警告。此限制不以自簽憑證掩飾；正式公開發行前仍需配置受信任的簽章憑證並重新驗證安裝與更新流程。
