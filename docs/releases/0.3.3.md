# 象棋 AI 分析講解 v0.3.3

v0.3.3 延續 v0.3.2 的 AI 教練、內建 Pikafish 與 GitHub Release 更新修正，並修正全新安裝時 NSIS 模式頁的錯誤提示。

## 主要改善

- 修正電腦上已沒有 App 安裝目錄與 App GUID 登錄時，安裝精靈仍顯示「已有每位使用者安裝／將重新安裝或升級」的問題。
- 安裝模式頁會同時核對 App 專屬 HKCU／HKLM `InstallLocation` 與實際主程式 EXE，再判斷全新安裝或升級；不掃描、不修改其他軟體登錄。
- 修正範圍只在 NSIS 安裝模式頁，不影響 electron-updater、App 核心或既有使用者資料。

## 同時包含的 v0.3.2 修正

- AI 教練不再把 Gemini 逾時誤稱為 Pikafish「引擎證據不足」，並會用使用者走棋理由與完整引擎主線比較實戰步和 AI 首選。
- 安裝版內附 Pikafish 執行檔、NNUE 權重與第三方授權／來源說明；乾淨安裝會自動登錄內建引擎。
- 安裝下載與自動更新統一使用 `enzohuang98-crypto/Reckoning` 的 GitHub Releases。
- Gemini API Key 不包含在安裝檔或 GitHub；必須由使用者在 App 內輸入，再由 Windows `safeStorage` 加密保存在本機。

## 已知限制

v0.3.3 尚未配置受信任 CA 核發的 Windows 程式碼簽章，Windows SmartScreen 可能顯示警告。這需要外部簽章憑證才能解除；本專案不以自簽憑證冒充公開信任。
