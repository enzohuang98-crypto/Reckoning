# 象棋 AI 分析講解 v0.3.4

v0.3.4 是乾淨安裝驗收後的首次設定修補版，包含 v0.3.2 的 AI 教練與內建 Pikafish 修正，以及 v0.3.3 的 NSIS 全新安裝辨識修正。

## 主要改善

- 修正首次設定內容高於視窗時，頁面受外層 `overflow: hidden` 裁切而無法捲動到 API Key 與「完成設定」按鈕的問題。
- 設定精靈現在固定使用視窗高度，內容超出時只在精靈內捲動；不改變主工作區、棋盤、引擎或 AI 運算流程。
- 新增針對設定精靈高度與捲動規則的回歸檢查。

## 安裝與資料

- 安裝版仍內附 Pikafish 執行檔、NNUE 權重及其授權與來源說明。
- Gemini API Key 不包含在安裝檔或 GitHub；必須由使用者在 App 內輸入，再由 Windows `safeStorage` 加密保存在本機。

## 已知限制

v0.3.4 尚未配置受信任 CA 核發的 Windows 程式碼簽章，Windows SmartScreen 可能顯示警告。這需要外部簽章憑證才能解除；本專案不以自簽憑證冒充公開信任。
