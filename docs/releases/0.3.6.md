# 象棋 AI 分析講解 v0.3.6

v0.3.6 建立 GitHub 為唯一正式來源的交付流程，並讓已安裝 App 在啟動後自動取得後續 GitHub Release。

## 主要改善

- 明確禁止直接修改已安裝目錄；可交付變更必須經 Git 分支、commit、GitHub Pull Request 與 Release。
- 正式版啟動後自動檢查並下載新 Release，使用者正常關閉 App 後自動安裝。
- 更新來源固定為本專案的公開 GitHub Release，不在 App 或倉庫內嵌 GitHub 權杖。
- 新增回歸檢查，確保正式版維持自動下載與結束後安裝設定。

## 更新行為

- App 必須至少啟動一次且能連線 GitHub，才會檢查並下載更新。
- 下載完成後不會中斷正在進行的棋局；更新在使用者正常關閉 App 後安裝。
- 從 v0.3.5 升級到 v0.3.6 仍需在設定頁手動按一次下載與安裝；自 v0.3.6 起，後續 Release 改為自動下載。

## 不受影響的範圍

- 棋盤、前端工作區、Pikafish 運算、AI 教練 Prompt 與 Gemini 呼叫流程均未變更。
- API Key 與個人資料仍只保存在本機，不會同步或上傳到 GitHub。

## 已知限制

目前尚未配置受信任 CA 核發的 Windows 程式碼簽章，Windows SmartScreen 仍可能顯示警告。
