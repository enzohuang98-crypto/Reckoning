# 象棋 AI 分析講解 v0.3.5

v0.3.5 是 GitHub 乾淨安裝驗收後的 Windows 安裝生命週期修補版，包含先前的 AI 教練、內建 Pikafish、GitHub Release 更新、NSIS 全新安裝辨識與首次設定捲動修正。

## 主要改善

- 修正安裝檔已完成檔案與捷徑安裝，但 Windows 的 App 專屬安裝／解除安裝登錄可能缺失，導致「已安裝的應用程式」無法辨識且日後升級判斷失真的問題。
- 安裝流程會依使用者選擇明確寫入 HKCU 或 HKLM，並立即讀回核對安裝路徑與版本；若無法完成，安裝會失敗而不宣告成功。
- Release 現在會在乾淨的 Windows runner 實際執行靜默安裝，核對版本、內建 Pikafish、NNUE、更新設定、App 登錄、桌面與開始功能表捷徑，再執行解除安裝並確認全部清除。
- 自訂 NSIS include 改用不會和 electron-builder 內建 `installer.nsh` 同名的檔名，降低 include 名稱碰撞風險。

## 不受影響的範圍

- 棋盤、前端工作區、Pikafish 運算、AI 教練 Prompt 與 Gemini 呼叫流程均未變更。
- Gemini API Key 仍不包含在安裝檔或 GitHub；只能由使用者在 App 內輸入並由 Windows `safeStorage` 加密保存在本機。

## 已知限制

v0.3.5 尚未配置受信任 CA 核發的 Windows 程式碼簽章，Windows SmartScreen 可能顯示警告。這需要外部簽章憑證才能解除；本專案不以自簽憑證冒充公開信任。
